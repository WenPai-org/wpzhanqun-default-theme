# 文派站群默认主题（WPZhanqun Default Theme）

## 默认主题可让您为网络上的新网站设置自定义默认主题。

### 这很简单

启用后，访问 **网络管理 > 设置**，然后选择您希望在所有新创建的网站上首先显示的主题。

### 使用：

1. 转到**网络管理 > 设置**

2. 向下滚动到主题设置。

3. 选择您希望在所有新创建的站点上看到的主题。

![image](截图 4)

4. 点击**保存更改**
